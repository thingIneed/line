Description

"Timeline Story" is a exciting and uplifting project entirely developted using native After Effects tools. 
  • After Effects CS4 or above in Full HD ( 1920 X 1080 px ).
  • 100% created using After Effects. Everything is editable.
  • Project very well organized, file with comments and tips. Very easy and fast to customize!
  • Video Tutorial included showing how edit this template.
  • No extra Plugins required! Flares were made with Video Copilot Optical Flares plugin, however the file comes with pre rendered version and original plugin version, if you need change something.
  • 31 Photo/Video holders
  • Project length: 2:42 sec.
  • You can use with or without photo captions!
  • 5 resolutions available, ready to render!

    *FULL HD ( 1920 x 1080 )
    *HDV ( 1280×720 )
    *DV NTSC Widescreen ( 720×480 )
    *DV PAL Widescreen ( 720×576 )
    *HALF  ( 960 x 540 )

Additional Support

GENERAL
The main project are "Timeline_Story.aep" or "Timeline_Story_VC_PreRendered.aep"  inside of "Timeline_Story" folder. 
All precomps used in project are inside the main composition “Timeline_Story_(main)”.
 
FONTS

Font usage in project: ARIAL

 
REPLACE PHOTOS OR VIDEOS
In “Project window” under /Assets /media folder select the file to be changed “image_01.jpg”, “image_02.jpg”, “image_03.jpg”,...
Then right click and go to option Replace Footage/File ( Ctrl + H ) and choose your new file.
 
Other option is open the file “image_XX.jpg” in Adobe Photoshop and drag and drop your photo to a new layer. Just save file, and reload the file in the After Effects.
 
IMAGE SIZE ( AVATAR)
The default photo size is 500 x 500 px. Scale if necessary.
 
IMAGE SIZE ( MODULE IMAGES)
The default photo size is 1280 x 720 px. Scale if necessary.
 
IMAGE SIZE ( PROFILE IMAGES)
The default photo size is 1850 x 700 px. Scale if necessary.
 
EDIT TEXTS
To customize texts use the precomps, inside of Precomps / customize_here / _STEPS and follow all steps.
Open these precomps, select text layer and edit with text tool. Change sizes, fonts, colors! Stay tuned to the tips layer inside of these precomps about change texts.
 
 
ADD AUDIO
Use this link to purchase and download audio music file. After purchase and download the audio file, open Aep project, then DRAG AND DROP your purchased audio files to (main) composition.
 
 
RENDER
5 options for resolution are pre defined in Render Queue window. Just mark the desired resolution in the window render queue and set the target folder.
The file is ready to generate a QuickTime Animation compressor, change according to your need.
  • 01 / Timeline_Story_(main) - Full HD (1920x1080)
  • 02 / Timeline_Story_(HDV) - HD (1280 x 720)
  • 03 / Timeline_Story_(DV_NTSC) - DV Widescreen (720 x 480)
  • 04 / Timeline_Story_(DV_PAL) - DV Widescreen (720 x 576)
  • 05 / Timeline_Story_(half) - Half (960 x 540)    ***FASTER VERSION
 
TIPS
  • To verify the animated keyframes for each layer, select the layer you want and press “U”.
  • To enter in any precomp double click in precomp layer.
  • If you have any problems in your render, try the secret option in the preferences (hold shift when you click preferences).
    In secret tab, have a option “purge all after every XX frames”. Put number 30. It should work for most memory errors.
  • Use "+" or "-" to zoom in or out in your timeline.
